# Криптография

Репозиторий содержит реализации классических криптографических алгоритмов и протоколов на языке Python, выполненные в рамках учебного курса по криптографии.

## Что здесь есть

**Протоколы обмена и передачи ключей** — протокол Диффи-Хеллмана, протокол передачи ключей, экспоненциальный протокол распределения ключей.

**Протоколы аутентификации и идентификации** — двухпроходный и трёхпроходный протоколы идентификации, протокол Фиата-Шамира, аутентификация на основе асимметричного шифра, одноразовые пароли.

**Цифровые подписи** — RSA, Эль-Гамаля, Фиата-Шамира, групповая цифровая подпись.

**Шифрование** — RSA, Эль-Гамаль, криптосистема Рабина, схема Блома.

**Хэш-функции и коды аутентификации** — SHA, Стрибог (ГОСТ Р 34.11-2012), HMAC.

**Разное** — схема разделения секрета Шамира, алгоритмы факторизации (ро-алгоритм Полларда, алгоритм p−1 Полларда), вспомогательные функции модульной арифметики, Base32/Base64.

## Технологии

Python 3, без сторонних библиотек.
