# Криптография

Коллекция лабораторных работ и реализаций по криптографии, выполненных в рамках университетской программы. Все лабы написаны на **Python**, без сторонних библиотек.

---

## Структура репозитория

### 7-й семестр - Классические и симметричные шифры

| Папка | Описание |
|-------|---------|
| `DES` | Реализация блочного шифра DES |
| `Анализ шифра Виженера` | Анализ и криптоанализ шифра Виженера |
| `Исследование статистических свойств текстов` | Исследование статистических свойств текстов |
| `Магма (ГОСТ Р 34.12-2015)` | Реализация российского блочного шифра Магма (ГОСТ Р 34.12-2015) |
| `Реализация шифра гаммирования` | Реализация шифра гаммирования (потоковый шифр) |
| `Шифр перестановки` | Реализация шифра перестановки |
| `Шифр простой замены` | Реализация шифра простой замены |

### Асимметричная криптография и протоколы обмена ключами

| Папка / Файл | Описание |
|--------------|---------|
| `Diffie-Hellman Protocol` | Протокол обмена ключами Диффи-Хеллмана |
| `Exponential key distribution protocol` | Протокол экспоненциального распределения ключей |
| `Key transfer protocol` | Протокол передачи ключей |
| `Blohm's scheme` | Схема предраспределения ключей Блома |
| `rsa.py` | Алгоритм шифрования RSA |
| `ElGamal.py` | Шифрование Эль-Гамаля |
| `Rabin.py` | Криптосистема Рабина |

### Цифровые подписи и аутентификация

| Папка / Файл | Описание |
|--------------|---------|
| `RSA digital signature` | Цифровая подпись на основе RSA |
| `El Gamal's digital signature` | Цифровая подпись Эль-Гамаля |
| `Fiat-Shamir's digital signature` | Цифровая подпись Фиата-Шамира |
| `Group digital signature` | Схема групповой цифровой подписи |
| `The Fiat-Shamir identification Protocol` | Протокол идентификации Фиата-Шамира (с нулевым разглашением) |
| `An authentication protocol based on an asymmetric cipher` | Протокол аутентификации на основе асимметричного шифра |
| `Two-pass identification protocol` | Двухпроходный протокол идентификации |
| `Three-pass identification protocol` | Трёхпроходный протокол идентификации |
| `One-time passwords` | Схема одноразовых паролей (OTP) |
